Projects

CentDash (Restarted: In Progress)

CentDash is a machine learning system for personal finance management that focuses on two core problems: predicting monthly spending patterns and detecting fraudulent/anomalous transactions in real-time. The platform combines a PyTorch LSTM neural network for time series spending forecasting with a scikit-learn Isolation Forest model for unsupervised anomaly detection. It demonstrates essential ML engineering practices in a production-ready FastAPI backend integrated with a modern React frontend, all running locally without requiring heavy infrastructure.

Tools being used: Flask, Next.js, React, TypeScript, Tailwind CSS, Plotly.js, SQLAlchemy, SQLite, Pydantic, PyTorch, Scikit-learn, NumPy, Pandas, PyTorch Lightning, Poetry, Pytest, Black, Ruff, VSCode, Docker.

RealEstate.AI (Restarted: In Progress) | Link

RealEstate.AI is a real estate analytics platform that combines machine learning with practical market insights. The system uses XGBoost gradient boosting for rapid property valuation predictions and Spring Boot 4 with Spring AI for intelligent real estate recommendations. The architecture demonstrates a realistic multi-service setup: a Java REST API backend, a Python ML microservice for model inference, and a React frontend. All components run locally and communicate via HTTP APIs, showcasing how modern AI systems integrate multiple languages and frameworks.

Tools used:Spring Boot, React, TypeScript, Tailwind CSS, Plotly.js, Axios, Spring AI, H2, Spring Data JPA, Swagger, Tomcat, FastAPI, XGBoost, Scikit-learn, Pandas, Joblib, Java, Maven, Poetry, VSCode.

DimeBoard (Restarted: In Progress)

DimeBoard is a financial markets platform that combines sentiment analysis with basic market forecasting. Built on ASP.NET Core 9 with a Python ML service, the system analyzes financial sentiment from market data and provides trend-based market insights. The platform demonstrates NLP capabilities, multi-language architecture (C# + Python), and real-time data handling. While intentionally simplified to fit storage constraints, it showcases the architecture and patterns of modern fintech AI systems.

Tools being used: ASP.NET Core, React, TypeScript, Tailwind CSS, Plotly.js, D3.js, Socket.io, WebSocket, .NET, SignalR, Entity Framework, Swagger, Kestrel, FastAPI, Sentence-Transformers, Pandas, NumPy, Poetry, VSCode.
