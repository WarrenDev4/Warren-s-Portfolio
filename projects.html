Projects

CentDash (Restarted: In Progress)

CentDash is a financial engineering system that applies machine learning to personal wealth optimization and risk management. It combines time series econometric modeling, portfolio theory, and advanced anomaly detection to help individuals make data-driven financial decisions. The platform demonstrates production-grade ML engineering applied to real financial problems: multi-asset portfolio optimization, spending forecasting with fiscal impact analysis, cash flow prediction under market uncertainty, and anomaly-driven fraud/risk detection. Built as an end-to-end ML system, it showcases financial domain expertise alongside robust ML operations practices.

Tools being used: Next.js, React 19, Tailwind CSS, Flask, FastAPI, PyTorch, Lightning AI, MLflow, Vector DB, Apache Airflow, Data: PostgreSQL, DuckDB, AWS,(SageMaker for experiments, Lambda for inference).

RealEstate.AI (Restarted: In Progress) | Link

RealEstate.AI is a production-grade machine learning system for real estate market analysis. It demonstrates data ingestion pipelines, feature engineering at scale, multi-model ensemble forecasting, and deployment of ML APIs in a microservices architecture. The system predicts property valuations, identifies investment opportunities, and forecasts neighborhood appreciation using structured and unstructured data.

Tools used: Spring Boot 4.0, Spring AI, LangChain4j, FastAPI (Python), React 19, TypeScript, XGBoost, LightGBM, CatBoost Apache Spark, Pandas, Polars Apache Airflow PostgreSQL, Prometheus + Grafana + ModelDB, LangChain + Pinecone, Docker + Kubernetes, AWS (EC2, SageMaker, S3, RDS)

DimeBoard (Restarted: In Progress)

DimeBoard is a sophisticated AI research system that combines sentiment analysis, time series forecasting, and reinforcement learning for financial market analysis. It demonstrates advanced ML patterns including transfer learning, multi-task learning, model ensembling, and production deployment of complex models.

Tools being used: ASP.NET Core, SignalR, FastAPI (Python), LangChain4j (Java agents), React 19, D3.js (visualization) HuggingFace Transformers, + Fine-tuning, PyTorch Lightning, Reinforcement Learning: Stable-Baselines3, Ray RLlib, Pandas, Polars, TA-Lib Apache Kafka, Apache Flink, Triton Inference Server, Feast (live feature retrieval), Prometheus + Grafana + Seldon Core, Backtrader + MLflow, Database: TimescaleDB (OHLCV data) + PostgreSQL (metadata) Azure, Redis
